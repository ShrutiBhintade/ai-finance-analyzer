# Styles

Global styles, Tailwind configuration, and design tokens.

## Files
- `styles/globals.css` — Tailwind imports, CSS variables, base styles
- `styles/tailwind.css` — Tailwind v4 config (if separate)
- `styles/theme.css` — design tokens (colors, spacing, typography)

## Guidelines
- Use Tailwind utilities first
- CSS variables for theming (light/dark mode)
- Avoid arbitrary values — extend theme instead
